How we collect your data

GD Skillsetguessr does not collect, store, or share any personal data from visitors. It also does not use cookies. All data is stored entirely on the user's device using localStorage for the sole purpose of saving every round the user has played. This data can be cleared at any time via either your browser settings, or the "CLEAR ALL DATA" button located in the "My best scores" menu, accessible through the main menu as long as you have local storage, and is never sent to any external server.

Third-Party Services

However, it incorporates embedded autoplaying YouTube videos, which load as soon as you choose a dataset on the main menu. These are supplied by Google LLC, which may collect information such as IP addresses, device details, and more personal information about you. For more details on how Google treats your data, please read the Google Privacy Policy.

The website also utilizes the API of the Geometry Dash Demon Ladder and the All Rated Extreme Demons List, neither of which store any of your personal data, nor does GD Skillsetguessr ever send them any personal data.

However, to fetch this data, it is processed through Cloudflare, which is only ever used as a means to fetch data from the previously mentioned APIs. Cloudflare may temporarily access your IP address and other information it needs to fetch the data securely.

For more information on how Cloudflare treats your data, please read the Cloudflare Privacy Policy.
